# Lista de Tareas (To-Do List)

Este es un proyecto que realicé para usar lo que aprendí con C++

## Características

- Agregar nuevas tareas.
- Ver la lista de tareas actuales.
- Marcar tareas como completadas.
- Interfaz simple y fácil de usar en la línea de comandos.

## Requisitos

- Un compilador de C++ (como `g++`).

## Cómo ejecutar

1. Clona este repositorio:

2. Compila el programa:
    ```bash
    g++ main.cpp -o lista_tareas
    ```

3. Ejecuta el programa:
    ```bash
    ./lista_tareas
    ```
